100p 에 당첨되셨습니다! 
const roulette = new Roulette('.roulette_container', {
    "duration": 3500,
    "defaultSpin": 4,
    "offset": 45,
    "startPosition": 0,
    "exceptResult": [1]
},[
    { name: '꽝!', range: [0, 90] },
    { name: '500p', range: [90, 180] },
    { name: '100p', range: [180, 270] },
    { name: '200p', range: [270, 360] },
]);

Dev Note.

  • range기반 정답 범위 생성
  • duration : 전체 회전 시간
  • defaultSpin : 기본적으로 돌아가는 최소 바퀴 수
  • offset : 룰렛 이미지 핀이 안맞는 경우 밀어서 맞추는 용도로 사용(룰렛의 지름 계산 시 복잡도 감소)
  • startPosition : 원하는 시작 위치 설정
  • start(idx) : 인자 없으면 그냥 랜덤으로 룰렛 돌리기, result의 idx를 넣으면 해당 idx에서 멈추게
  • 룰렛이 돌아가는중에는 start 버튼 눌러도 동작 안하게
  • 다시 돌릴때는 룰렛 rotate값 0으로 초기화
  • 점점 느리게 멈추게 하기
  • 엔터, 스페이스바, ESC키로 화면 제어 가능하게
  • 특정값으로 룰렛을 멈춰야 할때는, 정해진 위치가 아닌 해당 영역내에서 랜덤으로 멈추게
  • 룰렛 이미지에 따른 위치조정용 offset 변수 추가
  • 룰렛 초기 위치 값 startPosition 변수 추가
  • 제외할 결과 exceptResult 배열 추가
